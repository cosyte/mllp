Edit `package.json` to populate the `exports` map, `main`, `module`, and `types` fields.

tsup with `entry: { index, 'testing/index', 'ack-from-hl7/index' }` and `format: ['esm', 'cjs']` produces:
- `dist/index.js` (ESM), `dist/index.cjs` (CJS)
- `dist/index.d.ts` (ESM types), `dist/index.d.cts` (CJS types)
- `dist/testing/index.js`, `dist/testing/index.cjs`, `dist/testing/index.d.ts`, `dist/testing/index.d.cts`
- `dist/ack-from-hl7/index.js`, `dist/ack-from-hl7/index.cjs`, `dist/ack-from-hl7/index.d.ts`, `dist/ack-from-hl7/index.d.cts`

Set these fields in package.json:

```json
"main": "./dist/index.cjs",
"module": "./dist/index.js",
"types": "./dist/index.d.ts",
"exports": {
  ".": {
    "import": {
      "types": "./dist/index.d.ts",
      "default": "./dist/index.js"
    },
    "require": {
      "types": "./dist/index.d.cts",
      "default": "./dist/index.cjs"
    }
  },
  "./testing": {
    "import": {
      "types": "./dist/testing/index.d.ts",
      "default": "./dist/testing/index.js"
    },
    "require": {
      "types": "./dist/testing/index.d.cts",
      "default": "./dist/testing/index.cjs"
    }
  },
  "./ack-from-hl7": {
    "import": {
      "types": "./dist/ack-from-hl7/index.d.ts",
      "default": "./dist/ack-from-hl7/index.js"
    },
    "require": {
      "types": "./dist/ack-from-hl7/index.d.cts",
      "default": "./dist/ack-from-hl7/index.cjs"
    }
  }
}
```

IMPORTANT:
- The `types` condition inside each `import`/`require` block (NOT at the top of the entry object) is the correct pattern for TypeScript 5 moduleResolution:bundler / NodeNext.
- Each subpath needs BOTH `import` and `require` blocks, each with their own `types` condition.
- Do NOT use `"node"` conditions — those are for environment selection, not module system.

After editing package.json, run the build:
```bash
pnpm build
```
