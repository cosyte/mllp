The server does NOT crash; the peer will timeout waiting for the ACK and may retry.

`_buildAutoAck` implementation notes:
- Use `Buffer.from(..., 'ascii')` — not `'utf8'`. MSH header is 7-bit ASCII.

- The string `'|^~\\&|'` — the field separator is literally `|^~\&` (HL7 v2 encoding characters); in a JS string literal that's `'|^~\\&|'`.
